Perkenalan Diri

Saya adalah seorang mahasiswa d3 teknik informatika di sebuah perguruan tinggi bernama Universitas AMIKOM Yogyakarta. Pada program studi yang saya ambil saya memilih konsentrasi web development. Saya mengenal beberapa cabang atau pilihan di bidang teknologi seperti halnya Front-End Web Development, Backend Development, Multi-Platform Development, Cloud Computing, Machine Learning, Artificial Intelligence, Augmented Reality hingga Big Data.

Selain menekuni konsentrasi web development di perguruan tinggi, saya juga menekuni backend development dan android development pada platform belajar / pelatihan online Dicoding. Salah satu hal yang membuat saya tertarik dengan pemrograman adalah peluang untuk kerja jarak jauh atau work from home yang menurut saya dapat meningkatkan fleksibilitas kegiatan keseharian saya. Selain itu, besarnya peluang karir di dunia pemrograman juga menjadi salahsatu alasan saya untuk berusaha menekuninya.
